PROBLEM:
========
Das Rating bei compyo.de (und alle Websites) schwankt zwischen 77-84%
OHNE erkennbare Veränderungen auf der Website.

Root Cause: 3 NICHT-DETERMINISTISCHE Faktoren
==============================================

1. ❌ MEHRFACHE INKOMPATIBLE SCORING-FORMELN
   - scanner.py Zeile 188: 100 - (critical*20 + warning*5)
   - engine.py Zeile 464: 100 - (total_risk/max_risk*100)
   → Unterschiedliche Ergebnisse für gleiche Issues!

2. ❌ TCF-BONUS VON EXTERNAL API ABHÄNGIG
   - scanner.py Zeile 191-192: +5 Punkte wenn TCF-API verfügbar
   → Netzwerkfehler = Kein Bonus = Anderer Score!

3. ❌ CACHE OHNE VERFALL
   - risk_calculator.py Zeile 15: Cache wird NIE gelöscht
   → Multi-Instance Deployments: Unterschiedliche gecachte Daten
   → Gleiche Website, unterschiedliche Antworten je nach Instanz!
